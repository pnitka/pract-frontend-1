# pract-frontend-1
Откройте ваш CSS файл
Найдите пустой блок на вашей странице, который нужно будет стилизовать с помощью фонового изображения
Используйте свойство background-image, чтобы добавить фоновое изображение.
Используйте свойства background-repeat, background-position и background-size, чтобы настроить фоновое изображение по своему усмотрению.
Убедитесь, что у вашего блока есть заданные размеры, чтобы фоновое изображение было видно. Пример: width: 300px; height: 200px;
После внесения всех изменений, сохраните ваш CSS файл.
Обновите страницу в браузере: Перейдите к вашему браузеру и обновите страницу, чтобы увидеть, как фоновое изображение выглядит в вашем блоке.
